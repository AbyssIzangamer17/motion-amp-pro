Especificaciones Técnicas

01 / Eulerian Magnification

Algoritmo basado en filtrado temporal IIR sobre una pirámide Laplaciana para aislar y amplificar micro-movimientos espaciales.

02 / Latencia Cero

Pipeline asíncrono optimizado con Python 3.12+ y OpenCV. Procesa vídeo HD en tiempo real sin retrasos perceptibles.

03 / Control HUD

Ajusta la frecuencia de banda, el factor Alpha y el brillo de la cámara al vuelo con un HUD profesional interactivo.
